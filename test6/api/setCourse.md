# 接口：searchTest  [返回](../README.md)

- 权限：
    学生、老师：必须先登录，才能进行设置地址

- 功能：
    选择课程

- API请求地址：
   接口基本地址/api/setCourse
  
- 请求方式 ：
    GET

- 请求参数说明:
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |COURSE_ID|课程的编号|
  |TERM_ID|上课学期的编号|
  |STUDENT_ID|学号或者工号|

- 返回实例：

        {
            "status": true,
            "info": null,
            "data": [
                {
                    "MSG":"选课成功！"
                }
            ]
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
  |MSG|返回设置成功的消息|
