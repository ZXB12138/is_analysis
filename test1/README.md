## 客户维修服务流程
```
@startuml
|客户|
start
:申请服务;
|业务经理|
if (是新客户吗?) then(是)
    :登记客户信息;
else(无)
endif
:上门勘察;
:制定方案;
|客户|
if (满意吗？) then(否)
    stop
else(是)
    |业务经理|
    :签订服务合同;
    fork
        :安排工人;
    fork again
        :安排材料;
    fork end
    :填写派工单;
    |工人|
    :领取材料;
    :上门服务;
    |客户|
    :验收并填写反馈意见;
    |业务经理|
    :交回派工单;
    |财务人员|
    :结算收款;
    stop
@enduml
```


![客户维修服务流程](客户维修服务流程.jpg)



客户申请服务，业务经理看是否是新客户，新客户需登记客户信息，然后上门勘察，制订方案，客服是否满意，满意则签订服务合同，业务经理再分别安排工人和材料，再填写派工单，工人领取材料然后上门服务，客户验收并填写反馈意见，业务经理交回派工单，财务人员结算收款。
